# Pipeline Uzun Vadeli Yol Haritası

## Hedefler

- Satış tahminleme (forecasting) ve dönemsel hedef takibi
- Win/Loss analiz ekranı ve neden dağılımları
- Satış temsilcisi performans metrikleri (dönüşüm, cycle time)
- Stale uyarıları için e-posta/SMS entegrasyonu
- Pipeline stage yönetimi (admin panel)

## Önerilen Tasklar

1. **Sales Forecasting**
   - Dönem bazlı tahmin modeli (aylık/çeyreklik)
   - Forecast dashboard (commit/best case/pipeline)
   - Tahmin - gerçekleşen karşılaştırması

2. **Win/Loss Analysis**
   - Loss reason dağılımı
   - Rakip bazlı kayıp analizi
   - Satış süresi ve aşama bazlı kayıp nedenleri

3. **Sales Rep Performance**
   - Kişi bazlı dönüşüm oranları
   - Ortalama cycle time
   - Satış tutarı ve adet metrikleri

4. **Notification Dispatch**
   - Stale bildirimleri için e-posta/SMS gönderimi
   - Opt-in ayarları ve zamanlama

5. **Pipeline Stage Yönetimi**
   - Admin panelde stage ekleme/çıkarma
   - Stage olasılıkları ve sıralama yönetimi
